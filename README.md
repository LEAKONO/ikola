## ğŸš€ Getting Started

### Prerequisites
- Node.js 16+ and npm/yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/LEAKONO/ikola
   cd israel-ikola 
   ```
   
   ### Install dependencies

```bash
npm install
# or
yarn install
```
### Start development server

```bash
npm run dev
# or
yarn dev
```

## Customization
#### Update Content
- Personal Information: Edit Hero.jsx, Profile.jsx

- Education: Update Education.jsx with your credentials

- Experience: Modify Experience.jsx with your roles

- Contact Info: Update Contact.jsx and Footer.jsx

### Update Images
1. Profile Image: Replace public/images/profile.jpeg

2. Logos: Replace public/logo.jpeg and public/logo1.jpeg

3. Favicon: Update all favicon files in public/

### Colors & Styling
Edit the color scheme in:

- App.css - Global styles and animations

- Individual component files - Component-specific styles

- index.html - Theme color meta tags
